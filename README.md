# Check, Mate! ♟️

O **Check, Mate!** é um sistema simples e funcional de gestão de tarefas (To-Do List) desenvolvido para ajudar na organização da rotina acadêmica e pessoal. O projeto foca em mitigar a desorganização causada pela fragmentação de informações, centralizando tudo o que você precisa fazer em um único lugar.

## 🚀 Objetivo
O objetivo principal é oferecer uma solução de fácil acesso que reduza a carga cognitiva do usuário. Com uma interface minimalista, o sistema permite que estudantes e profissionais visualizem suas pendências e acompanhem seu progresso diário de forma instantânea.

## ✨ Funcionalidades
* **Cadastro de Tarefas:** Adicione rapidamente novas obrigações com validação de campo vazio.
* **Listagem Dinâmica:** Visualize todas as suas atividades em uma lista organizada em tempo real.
* **Controle de Status:** Marque tarefas como concluídas através de um botão visual, recebendo um feedback visual imediato (texto riscado).
* **Exclusão de Tarefas:** Remova itens da lista de forma simples.
* **Painel de Métricas:** Acompanhe em tempo real o número de tarefas pendentes e concluídas.
* **Interface Responsiva:** O layout adapta-se a diferentes tamanhos de tela para garantir uma boa experiência de uso.

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido utilizando tecnologias web puras (Vanilla), sem a necessidade de frameworks complexos, garantindo leveza e portabilidade:
* **HTML5:** Estrutura semântica da aplicação.
* **CSS3:** Estilização, layout Flexbox e feedbacks visuais.
* **JavaScript:** Lógica de manipulação do DOM e gestão de estado das tarefas.

## 📋 Como executar
Como o projeto utiliza tecnologias nativas do navegador, não é necessária nenhuma instalação complexa:
1. Faça o download ou clone este repositório.
2. Navegue até a pasta raiz `/check-mate`.
3. Abra o arquivo `index.html` em qualquer navegador web moderno (Chrome, Firefox, Edge, etc.).
